Procesador SASS

Instalación y Uso

SASS se instala con Node.js y npm. Ejecuta npm install -g sass para empezar a compilar tus archivos .scss a CSS.

Variables

Usa el símbolo $ para guardar colores y tamaños. Esto permite cambios globales rápidos y consistentes.

Operadores

Realiza cálculos matemáticos (+, -, *, /) directamente en tus estilos para medidas dinámicas y precisas.

Reglas Anidadas

Escribe selectores dentro de otros para reflejar la estructura HTML, manteniendo tu código limpio y organizado.

Mixins y Funciones

Reutiliza bloques de código con @mixin y realiza cálculos complejos con @function.

Directivas Avanzadas

Usa @import, @extend y condicionales para llevar tu CSS al siguiente nivel de eficiencia.

¿Cómo usar SASS en VS Code?

Instala la extensión "Live Sass Compiler". Crea tus archivos .scss y haz clic en "Watch Sass" para compilar automáticamente.
